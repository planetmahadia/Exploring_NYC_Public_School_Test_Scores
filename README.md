# DataCamp Project #2
Primarily using pandas to access a dataframe of NYC school math scores & manipulate, visualise and analyse the distribution of scores across schools.

Skills Used
- Python
- Pandas
- Data analysis
- Data visualisation

What I Did
- Cleaned the dataset
- Performed exploratory data analysis
- Summarised key insights

Provided with schools.csv & tasked with answering 3 key questions
1. Which 10 schools produce the best math results?
2. What are the top 10 performing schools based on the combined SAT scores?
3. Which single borough has the largest standard deviation in the combined SAT score?
